- Languages: Ruby
- Framework: Ruby on Rails
- Database: PostgreSQL

## Usage

Run the following commands:</br>
`git clone https://github.com/Hombre2014/Shopify_challenge`</br>
`cd Shopify_challenge`</br>
`bundle install`</br>
`rails s`</br>
Open a browser and go to: `https://localhost:3000` to view the app.
